## 1 Log

> *./sdkconfig -> `CONFIG_LOG_DEFAULT_LEVEL=3` Default `ESP_LOGI`*

|No.|Name|Remark|
|:---:|:---:|-----|
|1|`ESP_LOGE`|*error (lowest)*|
|2|`ESP_LOGW`|*warn*|
|3|`ESP_LOGI`|*info*|
|4|`ESP_LOGD`|*debug*|
|5|`ESP_LOGV`|*verbose (highest)*|
